# DigiCar (デジカー)

## 概要
DigiCarは[RumiCar](https://www.rumicar.com/ja/top-page/)互換のミニ四駆サイズの自動運転カーです。

DigiCarは[デジもく会](https://dezimoku.connpass.com/)が企画しています。
